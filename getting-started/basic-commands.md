# Basic commands



| Syntax            | Description                                                                             |
| ----------------- | --------------------------------------------------------------------------------------- |
| /spawnpoint       | Sets the spawnpoint of the player on the current location, adds player spawn to BlueMap |
| /togglehud on/off | Toggles the HUD, showing coordinates and current server time                            |
